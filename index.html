Email Test

A pair of simple .net test sites that let you check if you can send email via a given server.

Solution is for VS2013.

Two web projects are included (one MVC there asp.net), both offer the same functionality.

Usage;

  1. Select one of the projects as your start up project
  2. Set start up page / path (for asp.net project its /EmailTest.aspx for MVC it's Home/index)
  3. Enter the required fields (to, from, subject, body or body html, server)
  4. If you want to enter optional fields feel free, if values are given fields will not be including in the MailMessage or SMTPServer objects (except for port which is assumed to be 25).
  5. Click the send button, if the email is successful you will see an "email sent" message if it fails you'll see the asp.net "yellow screen of death"
